# 🚀 UpdateTrack

### Track. Check. Stay Updated.

UpdateTrack is a web-based application that helps users monitor software versions and check whether they are updated or outdated. It provides a simple and interactive interface similar to an app store.

---

## 🔍 Features

- 📦 App Store–style dashboard
- 🔎 Search applications easily
- ✅ Check if software is updated or outdated
- ⬇️ Direct update link for outdated applications
- ⭐ Add apps to favorites
- ❌ Remove apps from favorites
- 🌐 Dynamic data using JSON (auto updates)

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Browser Local Storage
- GitHub (for JSON data hosting)

---

## ⚙️ How It Works

1. The system fetches software data from a JSON file hosted on GitHub.
2. It compares the current version with the latest version.
3. Displays status:
   - ✅ Updated
   - ❌ Outdated
4. Provides update links for outdated apps.
5. Users can mark apps as favorites for quick access.
